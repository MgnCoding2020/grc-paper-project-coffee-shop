# Access Review Script

## Purpose
This script supports periodic access reviews by identifying inactive user accounts.

## Control Alignment
- AC-2 (Account Management)
- AC-6 (Least Privilege)

## Description
The script scans the access review CSV and flags accounts that have not logged in within 30 days.

## Usage
Run the script to generate a list of accounts requiring review or deactivation.

## Note
This script is a supporting tool and does not automate account changes.
